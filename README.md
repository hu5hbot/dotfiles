#dotfiles

I don't want to reveal much about my system but I want to archive the code that works for me for my future work/ease-of-access.

I'm leaving it here in case someone who needs assistance with a bit of code (that may not be well documented), has an easier time molding their desktop to their will.

Quick notes:
- Main becomes it's own branch per-year for archival purposes
- Archival code is present in files in comments (mostly .bashrc)
- I won't maintain a package-listing, as it's heavily volatile
- I prefer to set variables in .bashrc for global usage

OS: Arch
SH: Bash
WM: Ratpoison
TERM: XTerm
